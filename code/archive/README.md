## Archived Exploratory Scripts

These scripts are preserved for provenance but are no longer the canonical entry points for the REFER-II CLIF analysis.

Archived on 2026-05-12 during cleanup for a federated CLIF trajectory analysis workflow.

### Files

- `01_REFER_cohort_identification.R`: earlier ARF criteria-based cohort builder with broader inclusion/exclusion logic.
- `02_REFER_linkage_analysis.R`: broad linkage, history, outcome, and modeling script from an earlier REFER workflow.
- `02_REFER_II_analysis.r`: exploratory daily and hourly trajectory, exposure, and multinomial modeling code.
- `02_REFER_II_DTW.r`: exploratory multivariate DTW code with clinical and exposure trajectories.
- `02_REFER_II_DTW2.r`: expanded exploratory DTW pipeline with feature engineering, medoids, and plotting.
- `02_REFER_trajectories.r`: large exploratory trajectory script with several downstream figure and exposure analyses.

### Current Entry Points

Use these files from the parent `code/` directory instead:

- `01_cohort_identification.R`
- `02_trajectories.R`
